# GlassBlue Wallpaper

A calming blue glass-themed wallpaper for GNOME with elegant transparency effects. Features both light and dark variants that automatically adapt to your desktop theme.

## Features

- Light and dark mode support
- Glass-like visual effects
- Optimized for GNOME's wallpaper system
- High-quality image assets

## Installation

1. Ensure the script is executable (run `chmod +x install.sh` if needed)
2. Execute the installation script:
   ```bash
   ./install.sh
   ```

For global installation across all users, use sudo:

```bash
sudo ./install.sh
```

## Removal

To remove the wallpaper:

```bash
./install.sh -r
```

Or with sudo for system-wide removal:

```bash
sudo ./install.sh -r
```

## Files

- `GlassBlue.xml`: GNOME wallpaper configuration file
- `GlassBlueLight.png`: Light mode wallpaper image
- `GlassBlueDark.jpg`: Dark mode wallpaper image
- `install.sh`: Automated installation and removal script

## Requirements

- GNOME Desktop Environment
- Bash shell
