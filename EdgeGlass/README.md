# EdgeGlass Wallpaper

A modern glass-inspired wallpaper theme for GNOME with subtle edge effects. Includes both light and dark variants for automatic switching based on your system's theme.

## Features

- Light and dark mode support
- Seamless integration with GNOME background settings
- High-resolution images optimized for various screen sizes

## Installation

1. Make sure the script is executable (if not, run `chmod +x install.sh`)
2. Run the installation script:
   ```bash
   ./install.sh
   ```

For system-wide installation (affects all users), run with sudo:

```bash
sudo ./install.sh
```

## Removal

To uninstall the wallpaper:

```bash
./install.sh -r
```

Or with sudo if installed system-wide:

```bash
sudo ./install.sh -r
```

## Files

- `EdgeGlass.xml`: GNOME wallpaper configuration file
- `EdgeGlassLight.png`: Light mode wallpaper image
- `EdgeGlassDark.png`: Dark mode wallpaper image
- `install.sh`: Installation and removal script

## Requirements

- GNOME Desktop Environment
- Bash shell
