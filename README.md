This repository contains custom wallpaper themes for the GNOME desktop environment. Each theme includes light and dark variants with matching XML configuration files for seamless integration with GNOME's background settings.

## Available Wallpapers

- **[EdgeGlass](EdgeGlass/)**: A sleek glass-like wallpaper with edge effects.
- **[GlassBlue](GlassBlue/)**: A blue-tinted glass wallpaper theme.

## Installation

Each wallpaper comes with an automated installation script. Navigate to the desired wallpaper directory and run the installation script.

### Example:

```bash
cd EdgeGlass
./install.sh
```

For system-wide installation (requires root), run with `sudo`.

To remove a wallpaper:

```bash
./install.sh -r
```

## Requirements

- GNOME Desktop Environment
- Bash shell
